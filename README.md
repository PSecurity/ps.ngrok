# 🔐 PeekSecurity Installer

<p align="center">
  Script automatizado para instalação e configuração de ferramentas essenciais em <b>Termux</b> e <b>Kali Linux</b>.
</p>

---

## ⚡ Sobre o Projeto

O **PeekSecurity Installer** foi criado para facilitar o setup inicial de ambiente, automatizando processos como:

* Atualização do sistema
* Instalação de dependências
* Instalação do ngrok
* Configuração do authtoken
* Compatibilidade com múltiplos ambientes

Tudo isso com um menu interativo simples e direto.

---

## 🖥️ Ambientes Suportados

* 📱 Termux (Android)
* 💻 Kali Linux

---

## 🚀 Funcionalidades

✔ Detecção automática do ambiente
✔ Escolha manual de ambiente
✔ Instalação automática de dependências
✔ Download e configuração do ngrok
✔ Configuração opcional de authtoken
✔ Script leve e organizado

---

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/PSecurity/ps.ngrok
```

Acesse a pasta:

```bash
cd ps.ngrok
```

Dê permissão de execução:

```bash
chmod +x n26.sh
```

Execute o script:

```bash
./n26.sh
```

---

## 🔑 Configuração do Ngrok

Durante a execução, o script irá perguntar se você deseja configurar seu token.

Caso ainda não tenha um:

1. Acesse: https://dashboard.ngrok.com/get-started/your-authtoken
2. Copie seu token
3. Cole quando solicitado no script
